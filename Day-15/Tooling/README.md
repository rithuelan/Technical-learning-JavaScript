# JavaScript Tooling Demo

This project demonstrates **modern JS tooling** using:

- 🧩 **npm** – Dependency manager  
- 🧶 **Yarn** – Alternative package manager  
- ✅ **ESLint** – Code linter  
- 🎨 **Prettier** – Code formatter  
- 📦 **Webpack** – Bundler

---

## 🚀 Setup

```bash
# 1️⃣ Install dependencies
npm install

# 2️⃣ Lint code
npm run lint

# 3️⃣ Format code
npm run format

# 4️⃣ Build bundle
npm run build

# 5️⃣ Run app
npm start
